# Plotly.js Custom Vizualization Project - Splunk SE Summer Internship 2018
The plotly.js visualization library packaged as a Splunk application to expose the charts via the Splunk Custom Visualization Framework


Tamar A. Zamba -  Lead Developer 
SE Intern, Plano, TX

Tyler Muth - Project Manager
Analytics Architect,  Washington, D.C. 

This modular visualization app includes:
1. Candlestick/OHLC Chart -  for Stocks and Financial Data. See [Candlestick/OHLC source code](https://github.com/tmuth/plotly_custom_viz_splunk/blob/master/appserver/static/visualizations/candlestick_chart/src/visualization_source.js)

2. Box Plot Chart - for displaying Statistical Data. See [BoxPlot source code](https://github.com/tmuth/plotly_custom_viz_splunk/blob/master/appserver/static/visualizations/boxplot/src/visualization_source.js)
